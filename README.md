## Começando
* Primeiro: iniciar o back-end
* Depois execute os comandos:  
<code>npm i</code><br>
<code>npm run dev</code>
* Seu sistema irá iniciar em <a href="http://localhost:5173 ">***http://localhost:5173*** </a>
